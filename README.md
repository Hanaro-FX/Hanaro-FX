# Hanaro-FX
[디지털 하나로 금융 서비스 개발 2기]-FX 시뮬레이션 프로젝트

![poster](https://github.com/Hanaro-FX/Hanaro-FX/assets/68561229/0b9b249c-555f-4e83-aa19-638b75861175)
> Hanaro-FX는 외화만으로 구성된 투자 포트폴리오를 손쉽게 구상하고 테스트해 볼 수 있는 서비스입니다.
<br/>

## 🖥️ Demo
**1️⃣ 포트폴리오 생성**
- 사용자는 최대 35가지의 다양한 외화 중 원하는 통화를 선택하고, 각 통화의 비중을 조절하여 다양한 투자 전략 포트폴리오를 만들 수 있습니다.
<br/>

![create](https://github.com/Hanaro-FX/Hanaro-FX/assets/68561229/8b173c5b-47ab-43b8-a4ac-9931be428c0b)
<br/>

**2️⃣ 포트폴리오 테스트**
- 사용자는 기간과 Rebalancing 주기를 설정하여 자신의 포트폴리오를 테스트해 보고, 최적의 투자 전략을 찾을 수 있습니다.
- 포트폴리오 테스트 결과를 그래프로 시각화하여 사용자가 직관적으로 투자 성과를 확인할 수 있습니다.
<br/>

![test](https://github.com/Hanaro-FX/Hanaro-FX/assets/68561229/36ee4ef0-4957-4f6c-861b-15b26df17e42)
<br/>
